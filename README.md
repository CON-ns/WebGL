# Astroを用いた開発環境です

## 出来ること
- scssコンパイル
- JSバンドル
- css,jsのリント
- css,jsファイルのminify
- コンポーネントの活用

## 使用方法
cd astro-project : プロジェクトディレクトリへ移動
npm install : パッケージのインストール
npm run dev : 開発を開始,サーバーの起動
npm run build : ビルドする(distに出力する)

コピペ用
cd astro-project
npm install
npm run dev
npm run build
